# Selected Ironhack laboratory work

This directory contains selected classroom exercises from my 2021 Ironhack Data Analytics bootcamp. I retain them as evidence of the subjects I studied, not as production-ready applications.

| Notebook | Subject |
|---|---|
| `notebooks/customer_analysis_final.ipynb` | Customer and marketing analysis |
| `notebooks/marketing_customer_analysis_3.ipynb` | Customer segmentation |
| `notebooks/nlp_lab.ipynb` | Introductory natural language processing |
| `notebooks/computer_vision_lab.ipynb` | Introductory computer vision |
| `notebooks/sqlpybank.ipynb` | Python and Structured Query Language integration |
| `notebooks/errorhandling.ipynb` | Python error handling |
| `notebooks/string_operations.ipynb` | Python string operations |

Additional Structured Query Language exercises are in [`sql/`](sql/). The complete original upload remains frozen in [`../archive/bootcamp-original/`](../archive/bootcamp-original/).
